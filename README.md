# I-Taskmonitor
Currently only for Windows
Made with Imgui in C++.
This is a Basic Task monitor, but unlike a sitting duck of standard taskmonitor partial towards the windows process. 
This will show you clear view and control atleast thats the aim.

Phase 1: (view, identify and display metrics)
Features:
1.Basic metrics of process - PID, CPU,MEMORY,DISK,NETWORK,POWER - Added

2.System Details - Added
(shows CPU, motherboard, Ram type,
ram size, Disk storage, GPU and VRAM)

3.LOC - Added 
(Location of Compute) - shows if instance is from sys32 or win(c:\\windows) 
(will not show any program location its to check if program is from sys32 or win or not)

4. Fine tune displaying usage metrics
   
Future Features:
Add - Add/Remove programs Tab
Add - Startup program tab
Add - Services tab
Add - Firewall Tab
Add fan speed and Cpu temrature
Add Hover location funtion (when hovering over the LOC column for each thread show loction of program in small txt under pointer)
